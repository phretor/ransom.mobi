Svpeng

It was born as a banking trojan, but in 2014 it has evolved to include ransomware capabilities.

Samples of this family are able to encrypt files and to ask for a ransom (about $200) in order to decrypt them. Moreover in some cases they check if the device contains a mobile banking app, communicating the result of this search to a remote C&C server.
